Welcome!

This repo contains 2 improved versions of RackerLabs canon-jenkins theme for Hudson / Jenkins.

Pictures for both themes will be here tomorrow

Usage:

  • Install the Simple Theme Plugin for Jenkins.
  • Navigate to Jenkins > Manage Jenkins > Configure System > Theme
  • Set URL of theme CSS and JS to one of URLs below:

'Canon Jenkins' theme

This is an improved version of RackerLabs Rackspace Canon theme.

Why improved?

RackerLab`s theme is very nice.

But RackerLab`s design masterpiece is completely inconvenient if you have more than 20 jobs in Jenkins workspace.

So I updated that theme to contain more info per square inch of my small notebook display.

Full list of improvements:

  • Company logo removed.
  • Header and footer strips height was reduced.
  • Replaced annoying 'Help us to localize Jenkins' link with 'Enable auto-refresh' link
  • Search field was reduced in size and moved back to header to free space a bit (as it is done in stock Jenkins theme).
  • Annoying links underlining was avoided for main page.
  • Cross-browser gradient was used for header and footer strips.
  • Played with fonts, margins and paddings to make all views both more compact and readable.
